## 快速測試流程

### TC-MIC: 麥克風測試 (5 分鐘)

1. **回聲測試**: 不戴耳機，開喇叭通話，確認無回聲
2. **降噪測試**: 播放背景音樂，確認 Agent 能識別語音
3. **增益測試**: 小聲/大聲說話，確認 Agent 都能識別

### TC-VAD: 語音偵測測試 (5 分鐘)

1. **靜音測試**: 沉默 5 秒，確認黃色波浪平穩
2. **開頭測試**: 快速說「你好」，確認「你」字完整
3. **停頓測試**: 說完話後停 0.5 秒，確認可繼續說

### TC-PTT: 對講機模式測試 (5 分鐘)

1. 點擊「自動偵測」切換為「對講機模式」
2. 不按按鈕說話 → 確認 Agent 無反應
3. 按住按鈕說話 → 確認按鈕變紅、Agent 有回應
4. 放開按鈕 → 確認錄音停止

### TC-AQ: 音質測試 (3 分鐘)

1. 連線後觀察狀態列，確認顯示「延遲: xxxms | 音質: 高」
2. 開啟 Chrome DevTools → Network → Throttle → Slow 3G
3. 觀察是否自動降級

### TC-NET: 網路測試 (5 分鐘)

1. 執行 `measureLatency()` 檢查延遲
2. 關閉網路 → 確認顯示「重連 (1/3)」
3. 恢復網路 → 確認自動恢復
